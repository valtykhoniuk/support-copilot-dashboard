# Support Copilot Dashboard

Chat + AI ops monitoring for the [FoxSchool Support Copilot](https://github.com/valtykhoniuk/support-copilot) RAG agent.

## Stack

- React 19 + TypeScript + Vite

## Local dev

```bash
npm install
npm run dev
```

Backend must be running separately (`support-copilot` on `http://localhost:8000`).
